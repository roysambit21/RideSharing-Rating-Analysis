# RideSharing Rating Analysis

## Project Overview
This project analyzes customer satisfaction in ride-sharing services using Python, statistical analysis, exploratory data analysis (EDA), and machine learning techniques.

The objective of the project is to identify the major factors influencing customer satisfaction and build predictive models capable of estimating Service_Rating levels based on ride-sharing experience variables.

---

## Techniques Used
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Hypothesis Testing
- Logistic Regression
- Decision Tree
- Random Forest
- Feature Importance Analysis

---

## Key Findings
- Driver_Rating and Ride_Cost were among the most influential variables affecting customer satisfaction.
- Random Forest achieved the highest predictive accuracy (~71%).
- Service quality factors appeared more influential than demographic variables.
